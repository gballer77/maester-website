---
title: Third-party docs via includes
description: Aggregate documentation from third-party repos that don't publish their own maester.yaml by declaring an includes list on each source.
categories: [Configuration, Recipes]
link: https://github.com/baller-software/maester/tree/main/examples/third-party-includes
---

```yaml
# citadel.yaml — pulling docs from third-party repos
schemaVersion: 1
sources:
  - name: astro
    url: https://github.com/withastro/docs
    ref: main
    includes:
      - 'src/content/docs/en/**/*.mdx'
      - 'README.md'
  - name: tailwind
    url: https://github.com/tailwindlabs/tailwindcss.com
    ref: main
    includes:
      - 'src/pages/docs/**/*.mdx'
```
